Steps to setup the project:
1. Create a folder ECOMMERCE.
2. First in terminal run npm init.This will create pacakge.json file.
3. Add followimg dependencies : express, body-parser, sequelize, mysql2.
    this will add pacakge-lock.json file in the project.    
4. Make project configurable :
    a. add .env file // local environment - dev environment.
    b. in .env file mention port number PORT = 8080.
    c. in pacakge.json rename main file as server.js i.e. { "main": "server.js" } // optional step
5. Make a configs(configuration) folder.
6. Make a server.config.js file inside configs folder. This will contain all server's configuration details.
7. Create a server.js file in ECOMMERCE folder.
